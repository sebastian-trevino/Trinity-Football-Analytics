## Data

### sample_data.csv
Anonymized sample file showing the column structure of raw Hudl game exports.
Not real game data — provided for schema reference only.

### TUFB_EPA_Analysis.xlsx *(not included)*
The full engineered dataset output by `notebooks/03_ep_model.ipynb`.
Contains ~10 years of Trinity game data and is proprietary to Trinity University Athletics.

To run `dashboards/EPA_TUFB_Analysis_v001.ipynb`, generate this file by running
notebooks 01 through 03 first, then update the file path in the analysis script
to point to your Google Drive location.
